## Quick Start (2 steps!)

### Step 1: Add the workflow to your project

1. Create a `.antigravity` folder in your project root (if it doesn't exist)
2. Create a `workflows` subfolder inside it
3. Copy `setup-framework.md` into `.antigravity/workflows/`

```bash
mkdir -p .antigravity/workflows
cp setup-framework.md .antigravity/workflows/
```

### Step 2: Start the workflow in Antigravity

In Antigravity, simply say:

```
"Start the framework setup workflow"
```

or

```
"Let's set up the Universal App Framework"
```

## Troubleshooting

### "I don't see the workflow in Antigravity"
- Make sure the file is in `.antigravity/workflows/setup-framework.md`
- Restart Antigravity
- Or just start chatting - I can guide you without the formal workflow!

### "I want to start over"
- No problem! Just say "Let's restart the setup"
- I can clean up and start fresh

### "I only want to do part of the setup"
- Absolutely! Tell me which phases you want
- You can skip any phase

### "Something didn't work"
- Tell me what happened
- I'll diagnose and fix it
- We can rollback and try again
